# Локальные пользователи (Админ)

Эта папка предназначена для фотографий администраторов, которые должны распознаваться **всегда**, даже без интернет-соединения.

## Как добавить администратора:

1. Сделайте фото лица администратора (хорошее качество, четкое лицо)
2. Назовите файл по имени пользователя, например: `admin.jpg` или `ivan.jpg`
3. Поместите файл в эту папку `local_users/`
4. Перезапустите программу

## Поддерживаемые форматы:

- `.jpg` / `.jpeg`
- `.png`
- `.bmp`

## Примеры:

```
local_users/
├── admin.jpg          # Будет распознаваться как "admin"
├── security.jpg       # Будет распознаваться как "security"
└── manager.png        # Будет распознаваться как "manager"
```

## Важно:

- ✓ Локальные пользователи работают **без интернета**
- ✓ Не требуют синхронизации с сервером
- ✓ Не имеют ограничений по времени (expires_at)
- ✓ Не имеют расписания доступа (access_windows)
- ✓ События не отправляются на сервер
- ⚠️ Имя файла = имя пользователя в логах
- ⚠️ Одно фото = один пользователь

## Примечание:

Эта функция предназначена для **экстренного доступа** и **тестирования без сервера**.

Для полноценного управления пользователями используйте облачную синхронизацию.
